## 1. 模块概述

本文档定义兵牌（Unit Token）系统的完整设计，覆盖三支参战力量：**国军（中央军/地方军）、日军、八路军**。  
历史背景基准：**淞沪会战（1937年8-11月）**，兼顾华北战场八路军编制。

游戏为**即时战略可暂停**模式，以**天**为基础时间单位，无回合概念。兵牌是玩家在六边形棋盘上操控的战术单元，每张兵牌代表**团级作战单位**。

## 2. 编制结构

### 2.1 国军师（一师三团制）

每支国军部队以**师**为玩家操作单元，由 **1张师部牌 + 3张团牌** 组成，共4枚兵牌。

```
师部（HQ）
 ├── 步兵团 × 3
 └── 炮兵营（附属师部，通过师部技能体现）
```

| 兵牌类型 | 历史原型 | 代表兵力 | 战场标识 |
|----------|----------|----------|----------|
| 师部 | 第88师/第87师师部 | 约1,000人（指挥/炮兵/勤务） | 圆圈内「师」字 |
| 步兵团 | 各团（如262团、263团、264团） | 约2,750人 | 圆圈内「团」字 + 编号 |

> **精锐中央军**（88师、87师、36师）与**地方杂牌军**使用同一编制结构，数值不同，详见第4节。

---

### 2.2 日军师团（三联队制）

每支日军部队以**师团**为玩家操作单元，由 **1张师团司令部牌 + 3张联队牌** 组成，共4枚兵牌。

```
师团司令部（HQ）
 ├── 步兵联队 × 3
 ├── 骑兵联队（简化为司令部技能）
 └── 炮兵联队（简化为司令部技能）
```

| 兵牌类型 | 历史原型 | 代表兵力 | 战场标识 |
|----------|----------|----------|----------|
| 师团司令部 | 第3师团/第16师团司令部 | 约2,000人（指挥/炮/骑） | 圆圈内「司」字 |
| 步兵联队 | 各联队 | 约3,800人 | 圆圈内「联」字 + 编号 |

---

### 2.3 八路军师（一师四团制）

八路军以**师**为玩家操作单元，由 **1张师部牌 + 4张团牌** 组成，共5枚兵牌。  
历史原型：115师/120师/129师，每师编制约15,000人，强调机动与游击。

```
师部（HQ）
 ├── 步兵团 × 4
```

| 兵牌类型 | 历史原型 | 代表兵力 | 战场标识 |
|----------|----------|----------|----------|
| 师部 | 115师/120师/129师师部 | 约500人 | 圆圈内「师」字 |
| 步兵团 | 各团（如685团、686团、687团、688团） | 约1,500–2,000人 | 圆圈内「团」字 + 编号 |

## 3. 兵牌核心属性

```csharp
// TBS.Contracts — 兵牌数据契约
public interface IUnitToken
{
    // 基础识别
    string UnitId { get; }
    Faction Faction { get; }          // 国军/日军/八路军
    UnitTier Tier { get; }            // 师部/团/联队
    UnitGrade Grade { get; }          // 精锐/正规/杂牌/游击

    // 战斗属性
    int AttackPower { get; }          // 攻击力（0-10）
    int DefensePower { get; }         // 防御力（0-10）
    float MoveSpeedKmPerDay { get; }  // 行军速度（公里/天，即时制）
    int Firepower { get; }            // 火力（远程压制，0-10）

    // 资源属性
    int Strength { get; }             // 兵力值（0-5，满5=满编）
    int Morale { get; }               // 士气（0-100）
    int Supply { get; }               // 补给（0-5，每天消耗）

    // 状态
    UnitState State { get; }

    // 工事
    int FortificationLevel { get; }   // 工事等级（0-4）
    float FortificationProgress { get; } // 当前等级建设进度（0-1）
}
```

### 3.1 属性说明

| 属性 | 范围 | 说明 |
|------|------|------|
| **兵力值（Strength）** | 0 – 5 | 每格=满编20%；降至0时兵牌从棋盘移除（成建制歼灭） |
| **士气（Morale）** | 0 – 100 | 驱动状态切换；低于阈值触发状态恶化 |
| **补给（Supply）** | 0 – 5 | 补给线断绝后每天 -1；归零时攻防全面下降 |
| **攻击力** | 0 – 10 | 主动交战时的伤害基数 |
| **防御力** | 0 – 10 | 被攻击时的减伤基数（叠加工事加成） |
| **行军速度** | km/天 | 即时制下单位每天最大移动距离；地形/天气修正 |
| **火力** | 0 – 10 | 炮击/压制技能的强度基数 |
| **工事等级** | 0 – 4 | 0=无工事，4=永备工事；每级防御+1，需1天建设 |

## 5. 兵牌状态系统

状态由**士气**与**兵力值**共同驱动，在即时制下持续评估。  
不同等级部队拥有不同的**作战意志（Combat Will）**，决定崩溃临界值与士气恢复速度。

```csharp
public enum UnitState
{
    Inspired,     // 高昂：士气≥80 且 兵力≥4。攻击+2，速度+20%
    Normal,       // 正常：默认状态
    Suppressed,   // 压制：受到炮击/密集射击后触发（事件覆盖，约2-4小时）
    Shaken,       // 动摇：达到动摇临界。攻防各-1，速度-20%，每小时有概率自动后退
    Routed,       // 溃散：达到崩溃条件。无法执行命令，强制向后方撤退
    Recuperating, // 后撤整补：脱离前线+非战斗+有补给。自动恢复兵力与士气
}
```

> **兵力归零**：兵牌直接从棋盘移除，代表成建制歼灭，无过渡状态。

---

### 5.1 作战意志与崩溃临界值

**作战意志（Combat Will）** 是由兵牌等级（UnitGrade）决定的隐式属性组，包含三项阈值和一个恢复速率：

| 等级 | 动摇条件（→Shaken） | 崩溃条件（→Routed） | 士气恢复速度（/小时，非战斗休整） | 设计意图 |
|------|---------------------|---------------------|----------------------------------|----------|
| **国军·精锐** | 士气 < 40 | （兵力≤1 且 士气≤30）或 士气=0 | +2.5 | 德械精兵，即使伤亡惨重仍死战 |
| **国军·正规** | 士气 < 45 | 兵力≤1 或 士气≤15 | +1.5 | 有基础训练，但难以承受重大伤亡 |
| **国军·杂牌** | 士气 < 50 | 兵力≤2（≈50%）或 士气≤25 | +0.5 | 装备/训练不足，伤亡过半即崩 |
| **日军** | 士气 < 35 | （兵力≤1 且 士气≤20）或 士气=0 | +3.0 | 武士道精神，极难被单纯伤亡击溃 |
| **八路军·精锐** | 士气 < 40 | 兵力≤1 或 士气≤10 | +2.0 | 政委制度维系意志，但兵力有限 |
| **八路军·普通** | 士气 < 45 | 兵力≤1 或 士气≤20 | +1.5 | 游击队底子，兵力告急时撤退优先 |

> **关键设计说明**
> - 国军杂牌：兵力降至 **≤2（40%，接近50%）** 即触发溃散，无需士气判断
> - 国军精锐/日军：即使兵力仅剩1（20%），只要士气不跌至崩溃线，仍可坚持战斗
> - 日军崩溃条件最苛刻：须同时满足兵力极低 **且** 士气极低，或士气完全归零
> - 八路军溃散后优先执行战略撤退（游击战原则），不同于国军的无序溃逃

---

### 5.2 状态切换优先级

```
优先级 1（即时事件）：
  受到炮击/密集射击事件          → Suppressed（持续2-4小时，可叠加）

优先级 2（持续评估，每小时）：
  士气 ≥ 80 且 兵力 ≥ 4          → Inspired
  满足该等级崩溃条件              → Routed
  满足该等级动摇条件              → Shaken
  其余                            → Normal

优先级 3（直接移除）：
  兵力 = 0                        → 兵牌从棋盘移除（成建制歼灭）
```

### 5.3 状态效果汇总

| 状态 | 攻击 | 防御 | 速度 | 指令响应 | 可补充兵员 | 备注 |
|------|------|------|------|----------|------------|------|
| **高昂** | +2 | — | +20% | 正常 | 否（处于前线） | 进攻冲击力倍增 |
| **正常** | — | — | — | 正常 | 否（处于前线） | 基础数值 |
| **压制** | 无法主动攻击 | — | ×0.5 | 受限 | 否 | 事件触发，2-4小时消散 |
| **动摇** | -1 | -1 | -20% | 受限 | 否 | 每小时有概率自动后退1格 |
| **溃散** | 无法攻击 | -2 | ×0.7（强制后退） | 丧失 | 否（撤退途中） | 强制向后方移动，无法执行命令 |
| **后撤整补** | — | — | — | 正常 | **是** | 脱离前线后主动切换，见§5.6 |

### 5.4 士气恢复规则

**恢复生效条件**：单位处于非战斗状态（未受攻击、未主动攻击）且**持续超过1小时**。

恢复速度 = 基础恢复速度（见5.1表）× 环境修正

| 环境条件 | 士气恢复修正 |
|----------|-------------|
| 在师部/旅部指挥范围内 | ×2.0 |
| 补给充足（Supply ≥ 3） | ×1.5 |
| 补给断绝（Supply = 0） | ×0（无法自然恢复，每天额外-8） |
| 夜间休整（停止行军8小时以上） | ×1.5 |
| 溃散状态中 | ×0.3（极慢，需先撤离战场） |

**溃散恢复条件**：撤退至安全后方（距离最近敌军 ≥ 4格）且士气自然回升到该等级动摇临界值以上，方可解除溃散。

### 5.6 兵员补充（整补系统）

#### 触发条件

团牌同时满��以下三项，自动进入**后撤整补**状态：

1. **非战斗状态**：当前无交战（未受攻击、未主动攻击）
2. **脱离前线**：距离最近敌军兵牌 ≥ **4格**
3. **有补给来源**：自身 Supply ≥ 1，且与后方补给基地之间存在连通路径

> 玩家也可手动命令处于Normal/Shaken状态的团牌后撤至安全位置触发整补。

#### 补充规则

| 项目 | 规则 |
|------|------|
| **兵力恢复速度** | +1点 Strength / 天（每24小时，代表补充约20%满编兵力） |
| **补给消耗** | 每恢复1点 Strength 消耗 Supply 1点 |
| **上限** | Strength 最高恢复至满编（5点） |
| **士气联动** | 整补期间士气以基础速度×2.0恢复（等同在指挥范围内休整） |
| **中断条件** | 进入敌军4格范围内、或受到攻击，立即中断并退出整补状态 |

#### 各等级补充差异

| 等级 | 补充速度修正 | 设计意图 |
|------|-------------|----------|
| 国军·精锐 | ×1.0 | 后勤规范，按标准速度补充 |
| 国军·正规 | ×0.8 | 后勤能力一般 |
| 国军·杂牌 | ×0.5 | 后勤匮乏，补充极慢 |
| 日军 | ×1.2 | 后勤体系完善，补充快 |
| 八路军·精锐 | ×1.0 | 群众基础好，就地征募 |
| 八路军·普通 | ×0.8 | 装备补充有限 |

> **溃散状态的团牌**不可整补，须先完成撤退、解除溃散后方可启动整补。

## 6. 天时系统

游戏以**天**为基础推进，每天分为白天和黑夜两个阶段。

### 6.1 阶段划分

| 阶段 | 时长 | 特性 |
|------|------|------|
| 白天 | 约14小时 | 正常行军、战斗、侦察、工事建设 |
| 黑夜 | 约10小时 | 行军速度×0.5，侦察范围缩小；八路军夜间行军速度不减 |

### 6.2 天气类型与影响

| 天气 | 行军速度 | 空中支援 | 炮击精度 | 工事建设 | 士气（全天行军） |
|------|----------|----------|----------|----------|-----------------|
| **晴天** | ×1.0 | **可用** | 正常 | ×1.0 | — |
| **下雨** | ×0.5 | **不可用** | -30% | ×0.5 | -2 |

> **雨天优势**（八路军）：敌方炮击精度下降，适合发动伏击和穿插。  
> **晴天优势**（日军）：可呼叫空中支援，对无工事目标造成额外兵力伤害。

### 6.3 空中支援条件

- 仅晴天可用
- 目标格须在己方机场/舰载航母射程内（日军专属）
- 效果：目标兵牌兵力-1，士气-15，进入Suppressed状态

## 7. 防御工事系统

团牌**到达目标格后自动开始建设工事**，无需玩家手动指令。  
工事等级最高为4级，默认每升一级耗费1天（24小时）。

### 7.1 工事等级说明

| 等级 | 名称 | 防御加成 | 建设时间 | 描述 |
|------|------|----------|----------|------|
| 0 | 无工事 | +0 | — | 刚到达，无任何防御准备 |
| 1 | 散兵坑 | +1 | 1天 | 挖掘浅坑，基础掩体 |
| 2 | 野战壕 | +2 | 1天 | 连通战壕，防炮弹破片 |
| 3 | 加固阵地 | +3 | 1天 | 木石加固，抵御轻型炮击 |
| 4 | 永备工事 | +4 | 1天 | 混凝土工事，最强防御 |

> 总计最快**4天**可达满级工事。

### 7.2 工事建设规则

- **自动触发**：团牌停留（未移动）时自动推进工事进度
- **中断条件**：团牌发起主动攻击或离开当前格，工事进度**保留**（下次停留继续）
- **工事继承**：离开后返回同一格，继续之前进度（工事不会自动消失）
- **工事破坏**：受到3级以上炮击或重炮轰炸，工事等级-1（不低于0）
- **天气影响**：下雨天工事建设速度×0.5（泥泞）；晴天×1.0

### 7.3 工事与状态联动

| 工事等级 | 进入Suppressed状态所需炮击次数 |
|----------|-------------------------------|
| 0 | 1次 |
| 1 | 1次 |
| 2 | 2次 |
| 3 | 3次 |
| 4 | 4次（基本免疫轻型炮击） |

## 8. 师部/旅部特殊技能

技能冷却以**天**为单位（与即时制对应，可换算为小时）。  
师部/旅部无AP消耗，改为**冷却时间**控制使用频率。

### 8.1 指挥范围（被动）

| 势力 | 指挥范围（格） | 范围内效果 |
|------|---------------|------------|
| 国军精锐师部 | 3 | 下属团士气每天+5，防御+1 |
| 国军普通师部 | 2 | 下属团士气每天+3 |
| 日军师团司令部 | 4 | 下属联队士气每天+6，攻击+1 |
| 八路军师部 | 3 | 下属团士气每天+4，速度+10% |

> 师部被歼灭时，下属所有团牌立即 士气-20。

### 8.2 国军师部技能

#### 战时补给（Field Resupply）
- **效果**：目标团牌 Supply +2（上限5）
- **范围**：指挥范围内任意下属团牌
- **冷却**：3天
- **条件**：师部自身 Supply ≥ 2（消耗1点）
- **精锐加成**：效果改为 Supply +4

#### 炮兵支援（Artillery Support）
- **效果**：对指定格实施炮击；目标单位士气-15、兵力-1、进入Suppressed；雨天**不可用**
- **范围**：以师部为圆心5格内（不穿越高山）
- **冷却**：2天
- **条件**：师部 Supply ≥ 1（消耗1点）
- **精锐加成**：范围+1格，兵力伤害+1

#### 强袭冲锋（Assault Order）
- **效果**：目标团牌攻击力+3持续本次交战；攻击成功则敌方额外士气-10；攻击失败则己方额外士气-10
- **范围**：指挥范围内任意团牌
- **冷却**：3天

#### 构筑加固（Accelerated Fortification）
- **效果**：指挥范围内所有己方团牌工事建设速度×2，持续1天
- **范围**：指挥范围
- **冷却**：4天

### 8.3 日军师团司令部技能

#### 重炮轰炸（Heavy Artillery Barrage）
- **效果**：以目标格为圆心半径1格内所有单位 兵力-1、士气-20；工事等级-1；雨天**不可用**
- **范围**：7格内
- **冷却**：3天
- **条件**：Supply ≥ 2（消耗2点）

#### 空中支援（Air Support）
- **效果**：对指定格单位造成 兵力-1、士气-15；进入Suppressed
- **范围**：全图（受机场/舰队位置限制）
- **冷却**：1天
- **条件**：**仅晴天/多云可用**；Supply ≥ 1

#### 万岁冲锋（Banzai Charge）
- **效果**：目标联队攻击力+4，无视目标地形/工事防御加成；若此次交战兵力伤害≤1，该联队士气-25
- **范围**：指挥范围内任意联队
- **冷却**：4天

#### 侧翼包围（Flanking Maneuver）
- **效果**：本次作战周期内，对同一目标的第2/3支攻击单位各+2攻击力
- **范围**：全局
- **冷却**：2天

---

### 8.4 八路军师部技能

#### 隐蔽穿插（Concealed Movement）
- **效果**：目标团牌速度+50%，移动中不触发敌方反应攻击；到达目标格后进入隐蔽状态（本天不可被远程攻击）
- **范围**：指挥范围内任意团牌
- **冷却**：2天

#### 伏击部署（Ambush Setup）
- **效果**：在指定格布设伏击标记（敌方不可见）；敌军进入该格时触发：攻击+4，目标士气-20
- **范围**：指挥范围内任意未占领格
- **冷却**：3天
- **夜间加成**：夜间布设伏击标记不消耗冷却

#### 群众动员（Mass Mobilization）
- **效果**：目标团牌 Strength +1，同时揭开周围2格敌方迷雾（持续1天）
- **范围**：指挥范围内
- **冷却**：4天
- **条件**：目标格须为村庄/城镇地形

#### 化整为零（Disperse and Reform）
- **效果**：目标团牌进入分散状态1天：不可攻击，防御+4，免疫炮击；次日自动恢复正常
- **范围**：指挥范围内任意团牌
- **冷却**：3天

## 1. 模块概述

### 1.1 职责边界

地图系统负责游戏中六边形网格地图的生成、管理和查询，为其他系统提供统一的地图数据访问接口。

| 职责项 | 说明 | 边界外事项 |
|--------|------|------------|
| 网格生成 | 创建和管理六边形网格数据结构 | 不负责渲染表现（移交表现层） |
| 地形定义 | 定义地形类型及其属性 | 不负责地形视觉效果（移交表现层） |
| 坐标查询 | 提供坐标转换和邻接查询 | 不负责寻路算法（后续扩展） |
| 视野计算 | 提供视野/迷雾查询接口 | 不管理战争迷雾渲染（移交表现层） |

### 1.2 设计目标

- **可扩展性**：地形效果、地图尺寸、地图形状均可配置
- **性能优先**：支持大地图的快速查询和遍历
- **数据驱动**：地图配置通过JSON/ScriptableObject外部定义

### 2.2 类职责矩阵

| 类名 | 核心职责 | 依赖类 |
|------|----------|--------|
| `HexCoord` | 六边形坐标表示、坐标转换、邻接计算 | 无 |
| `HexTile` | 地块数据存储（地形、状态、单位引用） | HexCoord |
| `HexGrid` | 网格生成、存储管理、快速查询 | HexTile, HexCoord |
| `ITerrainEffect` | 地形效果抽象接口（契约） | HexTile |

## 3. 接口定义

### 3.1 对外暴露接口

```csharp
// 地图数据提供接口 - 供逻辑层使用
public interface IMapDataProvider
{
    // 获取指定坐标的地块
    HexTile GetTile(HexCoord coord);
    
    // 获取指定坐标范围内的所有地块
    IEnumerable<HexTile> GetTilesInRange(HexCoord center, int range);
    
    // 获取地图边界
    BoundsInt GetMapBounds();
}

// 地形查询接口 - 供战斗、移动系统使用
public interface ITerrainQuery
{
    // 获取地形移动消耗
    float GetMovementCost(HexCoord coord);
    
    // 获取地形防御加成
    float GetDefenseBonus(HexCoord coord);
    
    // 获取地形视野影响
    float GetVisibilityModifier(HexCoord coord);
}
```

### 3.2 扩展接口（模组支持）

```csharp
// 地形效果接口 - 可由模组扩展
public interface ITerrainEffect
{
    string TerrainType { get; }
    float MovementCost { get; }
    float DefenseBonus { get; }
    float VisibilityModifier { get; }
}
```

## 5. 坐标系统方案

### 5.1 坐标表示

采用**轴向坐标（Axial Coordinates）**作为主存储格式，立方体坐标（Cube Coordinates）用于计算。

```
     ____        ____
    /    \      /    \
   / q,r  \____/ q,r  \
   \ -1,0 /    \ 0,0  /
    \____/ q,r  \____/
    /    \ 0,-1 /    \
   / q,r  \____/ q,r  \
   \ -1,-1/    \ 0,-1 /
    \____/      \____/
```

### 5.2 坐标转换

- **轴向 → 偏移（Odd-R）**：用于渲染层显示
- **轴向 → 立方体**：用于距离计算
- **轴向 → 世界坐标**：用于Unity场景定位

## 7. 性能考虑

### 7.1 查询优化

- 使用 `Dictionary<HexCoord, HexTile>` 实现 O(1) 坐标查询
- 预计算邻接关系表避免重复计算

### 7.2 内存优化

- 地形类型使用ScriptableObject共享数据
- 地图数据支持分块加载（大地图场景）

## 8. 扩展性设计

### 8.1 支持的扩展点

| 扩展点 | 扩展方式 | 示例 |
|--------|----------|------|
| 地形效果 | 实现ITerrainEffect接口 | 新增沼泽地形效果 |
| 地图形状 | 继承HexGrid并覆写生成逻辑 | 圆形地图、不规则地图 |
| 地图数据源 | 实现IMapDataLoader接口 | 从远程加载地图 |

### 8.2 预留接口

- **寻路算法接口**：后续由独立文档定义（当前未实现）
- **视野/迷雾系统**：预留IVisibilityQuery接口，待视野系统设计后实现
